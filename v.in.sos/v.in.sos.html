DESCRIPTION

This module imports data from SOS server as a vector layer to GRASS GIS.

EXAMPLES

v.in.sos url=http://localhost/istsos/demo? output=humidity_points@user1 offering=GRIDA observed_properties='urn:ogc:def:parameter:x-istsos:1.0:meteo:air:humidity:relative' response_format='text/xml;subtype="om/1.0.0"'
