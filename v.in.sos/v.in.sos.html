DESCRIPTION

v.in.sos imports data from SOS server as a vector layer to GRASS GIS. Or it should, but currently it is importing only one layer...

EXAMPLES

Standard usage:

    v.in.sos url=http://localhost/istsos/demo? output=workshop \
    offering=workshop procedure=BELLINZONA \
    observed_properties=meteo:air:humidity:relative \
    event_time='2015-05-03 18:28:01+01:00/2015-05-03 19:33:01+01:00'
