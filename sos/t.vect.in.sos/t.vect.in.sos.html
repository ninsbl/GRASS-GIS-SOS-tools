DESCRIPTION

This module imports data from SOS server to GRASS GIS. They are imported as a layer for each timestamp.

DESCRIPTION

t.vect.in.sos imports data from SOS server to GRASS GIS. It creates new spatio-temporal map for each observed property of each procedure (registered from vector maps created as intermediates).

EXAMPLES

Standard usage:

    t.vect.in.sos url=http://localhost/istsos/demo? output=workshop \
    offering=workshop procedure=BELLINZONA \
    observed_properties=meteo:air:humidity:relative \
    event_time='2015-05-03 18:28:01+01:00/2015-05-03 19:33:01+01:00'
