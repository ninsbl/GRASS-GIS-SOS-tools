DESCRIPTION

v.in.sos imports data from SOS server as vector maps to GRASS GIS. It creates new raster map for each timestamp (containing all procedures within one offering and observed property for this timestamp).

User is also allowed to temporally aggregate the output by a specific temporal granularity. The temporal granularity can be seconds, minutes, hours, days, weeks, months or years. Mixing of granularities eg. 1 year, 3 months 5 days is not supported.

NOTES

To use this module, you need your OWSLib up-to-date. If you don't have it, please install it from its github repository.

Usage:

    r.in.sos url=http://localhost/istsos/demo? output=workshop \
    offering=temporary procedure=BELLINZONA,LOCARNO \
    observed_properties=rainfall \
    event_time='2015-05-03T18:28:01+01:00/2015-05-03T23:33:01+01:00' \
    granularity='1 hour' method='average' resolution=0.0001
