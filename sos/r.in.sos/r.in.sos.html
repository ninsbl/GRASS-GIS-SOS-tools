DESCRIPTION

r.in.sos imports data from SOS server as a raster maps to GRASS GIS. It creates new raster map for each timestamp.

EXAMPLES

Standard usage:

    r.in.sos url=http://localhost/istsos/demo? output=workshop \
    offering=workshop procedure=BELLINZONA \
    observed_properties=meteo:air:humidity:relative \
    event_time='2015-05-03 18:28:01+01:00/2015-05-03 19:33:01+01:00' -f
Output: 
    > g.list type=raster
     workshop_workshop_meteo_air_humidity_relative_t20150503T1830000100
     workshop_workshop_meteo_air_humidity_relative_t20150503T1840000100
     workshop_workshop_meteo_air_humidity_relative_t20150503T1850000100
     workshop_workshop_meteo_air_humidity_relative_t20150503T1900000100
     workshop_workshop_meteo_air_humidity_relative_t20150503T1910000100
     workshop_workshop_meteo_air_humidity_relative_t20150503T1920000100
     workshop_workshop_meteo_air_humidity_relative_t20150503T1930000100

Info about services:

    > r.in.sos url=http://localhost/istsos/demo? offering=workshop -p -v
