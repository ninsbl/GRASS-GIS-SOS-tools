DESCRIPTION

Convert a space time vector dataset into a space time raster dataset.

DESCRIPTION

t.vect.to.rast converts a space time vector dataset into a space time raster dataset. All vector maps in a space time vector dataset are passed to v.to.rast that converts them into vectors (points). Please refer to the v.to.rast documentation for a detailed description of the vector to raster conversion options. The new generated raster maps will be registered in the output space time raster dataset, using the same timestamps as their vector map layer origins.
